Introduction

Welcome to GeckoPlotter, a software designed to visualize and analyze High Performance Liquid Chromatography (HPLC) data efficiently. This tool provides an intuitive interface to manage your chromatographic data, customize plots, and (soon) perform detailed analyses.

Makes it easy to visualize your data quickly, make annotations, customize the color maps, and export them to various formats, GeckoPlotter improves your workflow, and allows you to make quality figures in no time.

Getting Started

Launching the Application

To start using GeckoPlotter, simply run the compiled application executable. The main window will appear, providing access to all primary functionalities.

Upon launching, you'll be greeted with the main interface, where you can begin selecting and loading your HPLC data files.

Main Interface Overview

The main interface of GeckoPlotter is designed for ease of use and quick access to essential features. Here's a breakdown of the primary components:

  • Menu Bar: Located at the top, it provides access to File, View, and Help menus for various operations like opening files, saving sessions, and accessing the User Guide.
  • Select Files Button: Allows you to browse and select multiple HPLC data files for analysis.
  • Plot Data Button: Opens the Plot Settings window where you can customize your plots before visualizing the data.
  • Load Session Button: Enables you to load a previously saved session, restoring all settings and loaded datasets.
  • Theme Toggle Button: Switches between Light and Dark themes to suit your preference.

Loading Data Files

Loading your HPLC data files is straightforward:

  1. Click on the "Select Files" button in the main interface.
  2. In the file dialog that appears, navigate to the directory containing your HPLC data files. Select one or multiple .txt files.
  3. Once selected, the application will display the number of files chosen. These files are now ready for plotting and analysis.
Note: Ensure that your data files are in the supported .txt format and contain the necessary columns: Time (min) and Value (mAU).

Plotting Data

After loading your data files, you can proceed to plot the chromatograms:

  1. Click on the "Plot Data" button to open the Plot Settings window.
  2. In the Plot Settings window, adjust your desired settings such as color maps, line thickness, X-axis range, and more.
  3. Select the datasets you wish to display by checking the corresponding boxes.
  4. Once all settings are configured, the plot will update automatically, reflecting your customizations.

Configuration Management

GeckoPlotter allows you to save and load your plot configurations, enabling you to maintain consistency across different sessions.

Saving a Session

  1. After configuring your plot settings and loading the desired datasets, navigate to File > Save Session in the menu bar.
  2. Choose a destination and filename for your session .json file.
  3. Click "Save". Your current configuration, including loaded datasets, plot settings, peaks, and annotations, will be saved.

Loading a Session

  1. To restore a previously saved session, navigate to File > Load Session.
  2. In the file dialog, select the desired session .json file.
  3. Click "Open". The application will automatically load all settings, datasets, and configurations from the session file.

Batch Processing

The Batch Processing feature allows you to handle multiple datasets simultaneously, streamlining your analysis workflow.

  1. Use the "Select Files" button to load multiple HPLC data files at once.
  2. Once loaded, all selected datasets will appear in the Dataset Selection panel within the Plot Settings window.
  3. You can choose to display them individually or combined in a single plot by toggling the "Plot Separately" option.

Peak Integration

Peak Integration is a critical feature for quantifying specific regions within your chromatograms.

  1. In the Plot Settings window, navigate to the Peak Settings section.
  2. Click on the "Add Peak" button to define a new peak.
  3. Specify the Minimum Time and Maximum Time boundaries for the peak you wish to integrate.
  4. Choose a color for the peak area using the Color Picker.
  5. Once defined, the area under the chromatogram curve within the specified time range will be highlighted, and the integrated area will be calculated and displayed in the console.
Note: Ensure that the defined time ranges correspond accurately to the peaks of interest in your data to obtain meaningful integration results.

Annotations

Annotations allow you to add textual information, arrows, or markers directly onto your plots, enhancing clarity and providing additional context.

  1. In the Plot Settings window, navigate to the Annotations section.
  2. Click on the "Add Annotation" button to create a new annotation.
  3. Configure the annotation by specifying:
    • Type: Choose between Text, Arrow, or Marker.
    • Target: Select the specific dataset to apply the annotation or choose All to apply it to all plots.
    • X and Y Coordinates: Define the exact location on the plot where the annotation will appear.
    • Text: Enter the content of the annotation.
    • Color: Choose a color that complements your plot for visibility.
  4. The annotation will appear on the plot based on the defined parameters.
Note: You can manage multiple annotations by adding or removing them as needed. Each annotation is customizable to fit your specific analysis requirements.

Customizing Plots

Customization options allow you to tailor your plots to meet your specific visualization needs.

Color Maps

Select from a variety of color maps to enhance the visual appeal of your plots:

  • Default: Uses the Viridis color map, known for its perceptual uniformity.
  • Alternative Options: Choose from a wide range of color maps such as Plasma, Inferno, Magma, and more to best represent your data.

Line Thickness

Adjust the thickness of the chromatogram lines to improve visibility:

  1. Navigate to the Line Thickness field in the Plot Settings window.
  2. Enter the desired line thickness value. Higher values result in thicker lines.
  3. Press Enter or click outside the field to apply the changes.

Axis Labels and Titles

Customize the labels and titles of your plots for better clarity:

  1. In the Additional Plot Customization section, modify the X-Axis Label and Y-Axis Label as needed.
  2. Adjust font sizes for axis labels, titles, and tick labels to enhance readability.

Legends

Enable or disable legends to either display or hide dataset labels on your plots:

  1. Toggle the "Show Legend" checkbox in the Plot Settings window.
  2. If enabled, legends will appear, indicating which color corresponds to which dataset.
  3. Adjust the legend font size for optimal visibility.

Exporting Plots

Export your customized plots in various formats for presentations, reports, or further analysis:

  1. After configuring your plot settings and ensuring the plot appears as desired, navigate to File > Export Plot in the menu bar.
  2. In the export dialog, select your preferred file format from the available options:
    • PNG: High-quality raster image suitable for presentations.
    • SVG: Scalable Vector Graphics for high-resolution printing and editing.
    • PDF: Document format ideal for sharing and embedding in reports.
    • EPS: Encapsulated PostScript for professional printing and publications.
  3. Choose the destination folder and filename for your exported plot.
  4. Specify the resolution (DPI) if prompted, especially for raster formats like PNG.
  5. Click "Save" to export your plot.
Tip: For best results, choose vector formats like SVG or PDF when possible, as they maintain quality at any scale.

Menu Options

The menu bar offers a range of options to manage files, settings, and access support resources:

File Menu

  • Open Files: Select and load HPLC data files into the application.
  • Save Session: Save the current state, including datasets and settings, for later use.
  • Load Session: Load a previously saved session to restore all configurations.
  • Save Settings: Save your plot customization settings independently of the session.
  • Load Settings: Load previously saved plot customization settings.
  • Export Plot: Export the current plot in your chosen format.
  • Exit: Close the application.

View Menu

  • Toggle Theme: Switch between Light and Dark themes to suit your environment.
  • Toggle Plot Settings: Show or hide the Plot Settings window for a cleaner interface.

Help Menu

  • User Guide: Access this comprehensive guide to learn about all features and functionalities.
  • Documentation: View detailed technical documentation for advanced usage.
  • Tutorials: Follow step-by-step tutorials to master the application.
  • About: Learn more about the application version, developer, and additional resources.

Theme and Appearance

Customize the appearance of GeckoPlotter to match your preferences or working environment:

  1. Locate the "Toggle Theme" button in the main interface or under the View Menu.
  2. Click the button to switch between Dark and Light themes.
  3. The application will immediately update its appearance, ensuring a comfortable viewing experience in different lighting conditions.
Note: The selected theme preference is saved with your session and will persist across different usage instances.
